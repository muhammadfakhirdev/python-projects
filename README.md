# Python Projects

This repository contains multiple Python mini-projects I have built for practice and learning.

## Projects

### 1. Splitwise Calculator
A program that helps split bills among multiple people by calculating equal or custom shares.

**Features:**
- Splits total bill among users
- Handles custom contributions
- Calculates who owes whom


### 2. Rock Paper Scissors Game

A simple command-line Python game where the user plays Rock Paper Scissors against the computer.

**Features:**
- User vs computer gameplay  
- Random computer choice generation  
- Win, lose, and draw logic  
- Simple interactive command-line interface
